# Paylocity GraphQL API

Paylocity is a cloud HR and payroll platform. The API covers employee records, payroll processing, time and attendance, benefits administration, expense management, performance reviews, and compliance reporting.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Paylocity REST API
**Documentation:** https://developer.paylocity.com/integrations/docs/getting-started
